## 参与贡献

我们欢迎各种形式的贡献——代码、文档、Issue 报告和 Feature 建议。你的参与让 PowerMem 变得更好。

### 如何贡献

1. Fork 仓库
2. 创建特性分支（`git checkout -b feature/my-feature`）
3. 提交更改
4. 运行测试（`make test`）
5. 提交 Pull Request

### 开发环境搭建

```bash
git clone https://github.com/oceanbase/powermem.git
cd powermem
pip install -e ".[dev]"
```

## 行为准则

PowerMem 社区致力于为所有参与者提供友好、包容和尊重的环境。我们期望所有社区成员遵循以下原则：

- **尊重** — 以尊严和尊重对待每个人
- **包容** — 欢迎多元化的视角和背景
- **建设性** — 提供有帮助的反馈和支持
- **专业** — 在所有互动中保持专业态度

违反行为准则的行为可向维护者报告。所有投诉将被及时审查和调查。

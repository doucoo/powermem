# 性能评测

PowerMem 在两个标准基准上评估：LOCOMO（长对话记忆）和 AppWorld（Agent 任务完成）。

## LOCOMO 数据集

LOCOMO 测试 Agent 从长对话历史中回答问题的能力。

| 指标 | PowerMem | 基线 | 变化 |
|------|----------|------|------|
| **准确率** | 87.79% | 52.9% | **+65.9%** |
| **检索 p95 延迟** | 1.44s | 17.12s | **-91.6%** |
| **每次查询 Token** | ~900 | ~26,000 | **-96.5%** |

**含义**：PowerMem Agent 回答更准确、快 12 倍、Token 消耗少 29 倍（相比基线的上下文窗口全量塞入方式）。

## AppWorld 数据集

AppWorld 测试 Agent 在模拟真实环境中的任务完成能力。

| 指标 | PowerMem | 基线 | 变化 |
|------|----------|------|------|
| **通过率** | 39% | 24% | **+62.5%** |
| **平均完成步数** | 6.2 | 9.5 | **-34.7%** |
| **总 Token** | 1.74M | 2.56M | **-32.0%** |

**含义**：使用 PowerMem 的 Agent 用更少步骤完成任务（减少试错），使用更少上下文。

## 为什么 PowerMem 获胜

1. **选择性检索** — 只加载相关记忆，而非整个历史
2. **结构化提取** — 事实经过预处理，不是原始聊天记录
3. **衰减与合并** — 过期记忆不污染检索结果
4. **混合检索** — 向量 + BM25 + 图谱覆盖单一方法遗漏的内容

# 核心特性

## 混合检索

PowerMem 在一个记忆层中融合三种检索策略：

- **向量检索** — 基于嵌入余弦距离的语义相似性
- **全文检索** — BM25 关键词匹配
- **图谱检索** — 实体关系遍历

结果通过互惠排名融合（RRF）或可配置加权打分合并，无需额外胶水代码。

## 智能记忆生命周期

记忆不是静态笔记堆。PowerMem 管理完整生命周期：

1. **提取** — LLM 从对话中识别关键事实和决策
2. **更新与合并** — 相似记忆自动合并
3. **强化** — 频繁访问的记忆增强（重要性分数提升）
4. **衰减** — 未使用的记忆按艾宾浩斯遗忘曲线衰减：`R = e^(-t/S)`

## 经验 + 技能蒸馏

两层自进化记忆系统：

| 层级 | 存储内容 | 示例 |
|------|----------|------|
| **经验** | 发生了什么、结果、上下文 | "部署到 prod-X 需要先连 VPN" |
| **技能** | 如何做事情、可复用的工作流 | "零停机部署的步骤" |

```python
memory.add_experience("用 Alembic 迁移数据库：务必先运行 autogenerate", user_id="alice")
memory.add_skill("数据库迁移：1) alembic autogenerate 2) 检查 diff 3) 执行", user_id="alice")
memory.distill_all()  # 批量蒸馏
```

## 多智能体隔离

三种作用域级别：**private**（仅 Agent）、**shared**（Agent 组）、**public**（跨组）。

## 用户画像

PowerMem 自动提取和维护用户画像：

- 语言偏好
- 沟通风格
- 领域专长
- 工作流模式

## 多模态支持

使用多种嵌入提供商处理文本、图片和音频：

- **文本**：OpenAI、Qwen、Gemini、HuggingFace、Ollama
- **图片**：Qwen 多模态、Gemini Vision
- **音频**：基于 Whisper 的转录
